# Teaching Brief: Algebra I Module 7 - Introduction to Functions

Source: `server/prisma/courses/math/algebra-i.json`
Objectives: Define a function; identify domain and range; evaluate functions; use function notation; distinguish functions from non-functions.
Assignment tie-in: Given 5 relations, determine if each is a function. Evaluate f(x) for 10 different inputs across 3 functions. State the domain and range of each.
