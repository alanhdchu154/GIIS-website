Welcome back. Algebra One, Module Seven — Introduction to Functions. A function is the most important machine in algebra. By the end of this lesson you can read function notation, evaluate a function, and tell a function apart from a relation that is not a function.

Picture a vending machine. You press B four. Out comes a granola bar — every time. You press B four again. Out comes a granola bar again. One button, one snack. That is a function — one input gives exactly one output. If pressing B four sometimes gave a granola bar and sometimes gave a soda, the machine would be broken — and it would not be a function.

Game plan. First, the rule — one input gives exactly one output. Second, function notation — f of x — read it as f at x, not f times x. Third, evaluate a function at several inputs and track the domain and range.

The core idea. A function is a rule that takes an input — usually called x — and produces exactly one output — usually called f of x. The set of allowed inputs is the domain. The set of outputs the function actually produces is the range. One x can only point at one y. If the same x points at two different outputs, the rule is a relation but not a function.

The killer mistake. Reading f of x as f times x. They look the same on paper but they mean different things. f of x is one symbol — the output of the function f when the input is x. The parentheses do not mean multiply here. They mean apply the rule. Whenever you see f of two, replace x with two everywhere in the rule — then simplify.

Evaluate f of x equals two x plus three. f of zero — substitute zero for x — two times zero plus three equals three. f of one — two times one plus three equals five. f of negative two — two times negative two plus three equals negative one. Three inputs, three outputs — and every input gives exactly one output. That is what makes it a function.

Evaluate g of x equals x squared minus four. g of three — three squared is nine, minus four equals five. g of negative three — negative three squared is also nine, minus four equals five. Different inputs, same output — that is allowed. A function can repeat outputs. What a function cannot do is give one input two different outputs.

Your turn. The function is h of x equals three x minus five. Evaluate h of zero, h of two, and h of negative one. Show each substitution before you simplify. Pause now. Solve it. Press play when you are ready.

Time. h of zero — three times zero minus five equals negative five. h of two — three times two minus five equals one. h of negative one — three times negative one minus five equals negative eight. Three inputs, three outputs, one rule. Notice the substitution step every time — that is what tells your teacher you understand the notation.

Now the domain and range. Domain is the set of allowed inputs. Range is the set of outputs that actually come out. For f of x equals two x plus three on the inputs zero, one, and negative two — the domain is the set negative two, zero, one — and the range is the set negative one, three, five. Same idea for any finite list — collect the x values and the f of x values into two sets.

Your turn. Decide whether each relation is a function. Relation A — the pairs one comma two, two comma four, three comma six. Relation B — the pairs one comma two, one comma five, two comma seven. Pause now. Decide. Press play when you are ready.

Time. Relation A — each input one, two, and three points at exactly one output. That is a function. Relation B — the input one points at two and also at five. Same input, two different outputs. That breaks the rule, so Relation B is not a function. The fast check — scan the x values. If any x repeats with a different y, it is not a function.

How the assignment uses this. Part A — five relations, decide which are functions and explain in one sentence why. Part B — three different functions, evaluate each one at ten inputs total, and show the substitution every time. Part C — for each of the three functions, list the domain and range of the inputs you tested.

Recap. A function is a rule where one input gives exactly one output. f of x means the output of f at input x — apply the rule, do not multiply. Domain is the set of inputs you tested. Range is the set of outputs that came out. If the same x ever pairs with two different y values, it is not a function.
