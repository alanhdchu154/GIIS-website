## What was KEPT despite reviewer flags, and why

- **Unit 7 "13–20%" weight (01_title).** Not in the reference but IS in
  the official AP CED. Real-world true; left as-is.
- **LUCA "three domains / shared core molecular machinery" framing
  (15_origin_of_life).** Not in the reference per Reviewer C, but
  real-world accurate and pedagogically critical. Kept and flagged here.
- **"Ribozymes are RNA enzymes" (15_origin_of_life).** Reviewer C noted
  this is not in the reference. Kept because RNA-world without ribozymes
  is empty — the catalytic + informational claim is the whole point.
- **Firefly flash example (05_prezygotic_isolation).** Confirmed in
  reference; kept.
- **K-Pg ~66 Mya and Chicxulub crater off Mexico (14_extinction).** ~66
  Mya is in the reference; Chicxulub is flagged as enrichment per
  Reviewer C's "keep only if flagged" guidance.
- **"Hair defines mammals" (11_clades_compare).** Reviewer C suggested
  dropping it; kept because it is the cleanest one-line example of a
  synapomorphy in introductory biology and the relativity caveat now
  carries most of the pedagogical weight. Substituting a different
  phylogeny example would have cost concreteness without paying back
  precision.
