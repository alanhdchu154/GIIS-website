# Teaching Brief: Algebra I Module 8 - Graphing Linear Functions

Source: `server/prisma/courses/math/algebra-i.json`
Objectives: Graph linear equations using a table of values, x/y-intercepts, and slope-intercept form; interpret graphs.
Assignment tie-in: Graph 8 linear equations (using Desmos or pencil/graph paper). For each, identify the x-intercept, y-intercept, and state whether the slope is positive, negative, zero, or undefined.
